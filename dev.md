# For devs

when working on TranscribeTools
- ensure you have the latest dependencies installed
- follow the coding style guide (use ruff :)
- test your changes thoroughly, add tests for new features
- keep the codebase clean and organized
- document your code for future reference

## working with UV 

`uv add` 
Add a new dependency

`uv build`
Build a package

`uv test` 

`uv run`
Run the current version of TranscribeTool

`uv publish`
Publish to pypi (you need to be an owner and have a pypi token)
